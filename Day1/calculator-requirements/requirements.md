# 要件定義書：電卓アプリ

## 1. アプリ概要

| 項目 | 内容 |
|------|------|
| アプリ名 | 電卓アプリ |
| 目的 | Webブラウザ上でシンプルな四則演算を素早く行えるようにする |
| ターゲット | 簡単な計算をしたい一般ユーザー |
| 方針 | 初心者が学習用に開発するため、複雑すぎない仕様とする |

---

## 2. 機能要件

### 必須機能

| 機能 | 説明 |
|------|------|
| 四則演算 | 加算（+）、減算（-）、乗算（×）、除算（÷）が行えること |
| クリア機能 | 表示や入力状態をリセットし、最初から入力できること |
| 数値入力（0〜9） | 0〜9の数字をボタンで入力できること |
| 表示 | 入力した数値・演算子・計算結果を画面上に表示すること |
| 実行 | 「=」などで計算を実行し、結果を表示すること |

---

## 3. 非機能要件

| 項目 | 内容 |
|------|------|
| レスポンス | ボタンを押してから**即座に画面が更新される**こと（体感で遅延がないこと） |
| コード品質 | JavaScriptのコードはシンプルで読みやすく記述する |
| 可読性 | 初心者でも理解できる構造にする（関数の役割を分ける、命名を分かりやすくする等） |
| 使いやすさ | ボタン配置や表示が分かりやすく、誤操作しにくいこと |

---

## 4. 画面構成（UIイメージ）

```
+------------------------------------------+
|              電卓アプリ                    |
+------------------------------------------+
|                                          |
|  [ 表示エリア（計算式・結果） ]           |
|  例: 12 + 34 = 46                        |
|                                          |
+------------------------------------------+
|  [ 7 ]  [ 8 ]  [ 9 ]  [ ÷ ]              |
|  [ 4 ]  [ 5 ]  [ 6 ]  [ × ]              |
|  [ 1 ]  [ 2 ]  [ 3 ]  [ - ]              |
|  [ 0 ]  [ C ]  [ = ]  [ + ]              |
+------------------------------------------+
```

- **表示エリア**: 現在の入力や計算結果を表示する（1行で十分）
- **数字ボタン**: 0〜9
- **演算子ボタン**: +、-、×、÷
- **実行ボタン**: =（等号）
- **クリアボタン**: C（Clear）で表示・状態をリセット

※ レイアウトは上記を参考にし、初心者が実装しやすいシンプルな構成とする。

---

## 5. 扱うデータ

| データ | 説明 | 保持場所 |
|--------|------|----------|
| 表示用テキスト | 画面上に表示する文字列（数値・演算子・結果） | 変数（例：`displayValue`） |
| 現在の入力値 | ユーザーが今入力している数値 | 変数（例：`currentValue`） |
| 前の値・演算子 | 「12 +」まで入力したときの 12 と + | 変数（必要に応じて） |
| 計算結果 | 「=」を押したときの結果 | 表示用変数に反映 |

- いずれも**メモリやDBには保存しない**（ページを開いている間だけ保持する一時データ）
- リロードやタブを閉じると消える仕様でよい

---

## 6. 前提条件・制約

| 項目 | 内容 |
|------|------|
| 技術スタック | **HTML / CSS / JavaScript のみ**で実装する（フレームワーク・ライブラリは使わない） |
| 動作環境 | 一般的なWebブラウザ（Chrome、Edge、Firefox、Safari等）で動作すること |
| ファイル | 1つのHTMLファイルにまとめてもよい。または HTML / CSS / JS を分離してもよい |
| 学習目的 | 初心者が読んで理解できる範囲の複雑さに留める |

---

## 7. スコープ外（今回やらないこと）

- **高度な計算**
  - 平方根、累乗、三角関数、対数など
- **計算履歴**
  - 履歴の保存・一覧表示・再計算
- **デザイン・演出**
  - 高度なアニメーション、複雑なレイアウト
- **その他**
  - キーボード入力対応（あれば便利だが必須としない）
  - 複数桁・小数点の細かい仕様は必須範囲内でシンプルに決める
